# Contribution Guidelines

Thank you for your interest in contributing to RepoDiscoverAI! We welcome contributions from anyone who wants to help improve this resource.

## How to Contribute

1.  Fork the repository.
2.  Create a new branch for your feature or bug fix.
3.  Add a new directory in to reflect to the area you would like to include. Add a new README.md file within the new directory to contain the relevant repo information
4.  Submit a pull request.

## Code of Conduct

This project follows a Code of Conduct. By participating, you are expected to uphold this code. Please report unacceptable behavior to [weisensopenbot@example.com](mailto:weisensopenbot@example.com).
