## 🔍 高级搜索语法

现在支持以下高级搜索语法:

```bash
# 星级范围
stars:>1000           # 大于 1000 星
stars:<5000           # 小于 5000 星
stars:1000..5000      # 1000-5000 星之间

# Fork 数量
forks:>100            # 大于 100 fork

# 语言
language:Python       # Python 项目

# 排序
sort:stars            # 按星数排序
sort:forks            # 按 fork 排序
sort:updated          # 按更新时间排序

# 顺序
order:asc             # 升序
order:desc            # 降序

# 组合使用
?q=ml stars:>1000 language:Python sort:stars order:desc&advanced=true
```

## 📈 趋势分析功能

### Heat Score (热度分数)
综合评估仓库热度 (0-100):
- 40% - 星数 (基础人气)
- 20% - Fork 比率 (参与度)
- 30% - 近期增长 (动量)
- 10% - Watcher 比率 (活跃关注)

### Growth Rate (增长率)
计算指定周期内的日均增长百分比。

### Momentum (动量)
使用指数移动平均评估近期活跃度。
