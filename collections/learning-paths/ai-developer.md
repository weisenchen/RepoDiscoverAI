# AI Developer 学习路径

**持续时间:** 6-12 个月  
**难度:** 中级  
**前置要求:** 基础编程知识

---

## 📚 路径概述

本学习路径将帮助你从编程基础成长为能够构建 AI 应用程序的开发者。涵盖 Python 编程、机器学习基础、深度学习框架和 LLM 应用开发。

---

## 🎯 阶段 1: Python 基础 (4-6 周)

### 学习目标
- 掌握 Python 基础语法
- 理解数据结构（列表、字典、集合）
- 学会函数和模块的使用
- 了解面向对象编程

### 推荐项目
1. **自动化脚本** - 文件处理、数据清洗
2. **Web 爬虫** - 使用 requests + BeautifulSoup
3. **数据分析小项目** - 使用 pandas 分析 CSV 数据

### 推荐仓库
- `automate-the-boring-stuff` - Python 自动化入门
- `python-patterns` - Python 设计模式

---

## 🎯 阶段 2: 机器学习基础 (8-10 周)

### 学习目标
- 理解机器学习基本概念
- 掌握 scikit-learn 库
- 学会数据预处理和特征工程
- 了解常见算法（回归、分类、聚类）

### 推荐项目
1. **房价预测** - 线性回归
2. **手写数字识别** - 使用 MNIST 数据集
3. **客户分群** - K-means 聚类

### 推荐仓库
- `scikit-learn/scikit-learn` - 机器学习库
- `jakevdp/PythonDataScienceHandbook` - 数据科学手册

---

## 🎯 阶段 3: 深度学习 (10-12 周)

### 学习目标
- 理解神经网络原理
- 掌握 PyTorch 或 TensorFlow
- 学会 CNN、RNN 等架构
- 了解迁移学习

### 推荐项目
1. **图像分类** - 使用预训练模型
2. **文本情感分析** - RNN/LSTM
3. **目标检测** - YOLO 或 Faster R-CNN

### 推荐仓库
- `pytorch/pytorch` - 深度学习框架
- `huggingface/transformers` - NLP 模型库
- `ultralytics/yolov5` - 目标检测

---

## 🎯 阶段 4: LLM 应用开发 (8-10 周)

### 学习目标
- 理解大语言模型原理
- 掌握 Prompt Engineering
- 学会使用 LangChain 等框架
- 能够构建 RAG 应用

### 推荐项目
1. **智能问答机器人** - 基于文档的 QA 系统
2. **代码助手** - 使用 Code LLM
3. **数据分析助手** - 自然语言查询数据

### 推荐仓库
- `langchain-ai/langchain` - LLM 应用框架
- `Shubhamsaboo/awesome-llm-apps` - LLM 应用合集
- `openai/openai-python` - OpenAI API

---

## 🏆 毕业项目

选择一个方向完成一个完整的项目：

1. **AI 助手** - 结合多个 AI 能力的综合应用
2. **行业解决方案** - 针对特定行业的 AI 应用
3. **开源贡献** - 为 AI 相关开源项目贡献代码

---

## 📖 持续学习

- 关注 arXiv 最新论文
- 参与 Kaggle 竞赛
- 贡献开源项目
- 参加 AI 社区活动
