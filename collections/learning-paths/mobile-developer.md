# Mobile Developer 学习路径

**持续时间:** 8-12 个月  
**难度:** 中级  
**前置要求:** 基础编程知识

---

## 📚 路径概述

本学习路径将帮助你成长为移动应用开发者。提供 iOS (Swift)、Android (Kotlin) 和跨平台 (Flutter/React Native) 三个方向的选择。

---

## 🎯 阶段 1: 移动开发基础 (6-8 周)

### 学习目标
- 理解移动应用架构
- 掌握至少一门移动语言
- 了解 UI/UX 设计原则
- 学会调试和测试

### 方向选择

#### iOS (Swift)
- Swift 语言基础
- Xcode 使用
- UIKit/SwiftUI

#### Android (Kotlin)
- Kotlin 语言基础
- Android Studio 使用
- Jetpack Compose

#### 跨平台
- Flutter (Dart) 或 React Native
- 跨平台架构理解

### 推荐项目
1. **Hello World 应用** - 熟悉开发环境
2. **简单计算器** - UI 和逻辑
3. **待办应用** - 本地数据存储

### 推荐仓库
- `apple/swift` - Swift 语言
- `JetBrains/kotlin` - Kotlin 语言
- `flutter/flutter` - 跨平台框架
- `facebook/react-native` - React Native

---

## 🎯 阶段 2: 核心功能开发 (8-10 周)

### 学习目标
- 掌握网络请求
- 学会本地数据存储
- 理解状态管理
- 了解导航和路由

### 推荐项目
1. **天气应用** - API 调用 + 展示
2. **新闻阅读器** - 列表 + 详情
3. **用户认证** - 登录注册流程

### 推荐仓库
- `square/retrofit` - Android HTTP 客户端
- `Alamofire/Alamofire` - iOS HTTP 客户端
- `hive/hive` - Flutter 本地存储
- `realm/realm-swift` - 数据库

---

## 🎯 阶段 3: 高级功能 (8-10 周)

### 学习目标
- 掌握推送通知
- 学会使用设备功能（相机、GPS）
- 理解后台任务
- 了解性能优化

### 推荐项目
1. **社交应用** - 图片上传 + 推送
2. **地图应用** - 定位 + 路线
3. **音乐播放器** - 后台播放

### 推荐仓库
- `googlemaps/ios` - Google Maps iOS
- `googlemaps/android` - Google Maps Android
- `firebase/firebase-ios-sdk` - Firebase iOS
- `firebase/firebase-android-sdk` - Firebase Android

---

## 🎯 阶段 4: 发布与运维 (4-6 周)

### 学习目标
- 理解应用商店审核
- 学会应用签名和发布
- 了解崩溃监控
- 掌握用户分析

### 推荐项目
1. **应用上架** - App Store / Google Play
2. **崩溃监控** - 集成 Sentry/Crashlytics
3. **数据分析** - 用户行为追踪

### 推荐仓库
- `getsentry/sentry` - 错误监控
- `firebase/firebase-ios-sdk` - 分析
- `mixpanel/mixpanel-iphone` - 用户分析

---

## 🏆 毕业项目

发布一个完整的移动应用：

1. **生产力工具** - 解决实际问题
2. **社交应用** - 用户互动功能
3. **游戏应用** - 休闲小游戏

---

## 📖 持续学习

- 关注平台更新（WWDC、Google I/O）
- 学习新框架和工具
- 参与开源项目
- 加入开发者社区
