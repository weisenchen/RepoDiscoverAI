# Web Developer 学习路径

**持续时间:** 6-9 个月  
**难度:** 入门级  
**前置要求:** 无

---

## 📚 路径概述

本学习路径将帮助你从零开始成长为全栈 Web 开发者。涵盖前端三件套、现代框架、后端开发和部署运维。

---

## 🎯 阶段 1: HTML/CSS/JavaScript 基础 (6-8 周)

### 学习目标
- 掌握 HTML 语义化标签
- 理解 CSS 布局（Flexbox、Grid）
- 学会 JavaScript 基础语法
- 了解 DOM 操作和事件处理

### 推荐项目
1. **个人主页** - HTML + CSS
2. **响应式博客** - 移动端适配
3. **交互式待办列表** - JavaScript DOM 操作

### 推荐仓库
- `twbs/bootstrap` - CSS 框架
- `tailwindlabs/tailwindcss` - 实用优先 CSS
- `30-seconds/30-seconds-of-code` - 代码片段

---

## 🎯 阶段 2: 前端框架 (8-10 周)

### 学习目标
- 掌握 React/Vue/Angular 其中之一
- 理解组件化开发
- 学会状态管理
- 了解路由和 API 调用

### 推荐项目
1. **电商网站** - 商品列表、购物车
2. **社交应用** - 用户动态、评论
3. **数据仪表盘** - 图表展示

### 推荐仓库
- `facebook/react` - React 框架
- `vuejs/core` - Vue 框架
- `reduxjs/redux` - 状态管理
- `vercel/next.js` - React 全栈框架

---

## 🎯 阶段 3: 后端开发 (8-10 周)

### 学习目标
- 掌握一门后端语言（Node.js/Python/Go）
- 理解 RESTful API 设计
- 学会数据库操作（SQL/NoSQL）
- 了解认证和授权

### 推荐项目
1. **博客 API** - CRUD 操作
2. **用户系统** - 注册、登录、JWT
3. **文件上传服务** - 云存储集成

### 推荐仓库
- `expressjs/express` - Node.js 框架
- `fastapi/fastapi` - Python API 框架
- `prisma/prisma` - 数据库 ORM
- `mongodb/mongodb` - NoSQL 数据库

---

## 🎯 阶段 4: 部署与运维 (4-6 周)

### 学习目标
- 理解 Docker 容器化
- 学会 CI/CD 流程
- 了解云服务（AWS/Vercel/Netlify）
- 掌握基础监控和日志

### 推荐项目
1. **Docker 化应用** - 容器部署
2. **GitHub Actions** - 自动测试部署
3. **监控仪表盘** - 性能监控

### 推荐仓库
- `docker/compose` - 容器编排
- `actions/github` - GitHub Actions
- `grafana/grafana` - 监控仪表盘
- `vercel/vercel` - 前端部署平台

---

## 🏆 毕业项目

构建一个完整的全栈应用：

1. **SaaS 产品** - 订阅制服务
2. **电商平台** - 完整的购物流程
3. **社交网络** - 用户互动功能

---

## 📖 持续学习

- 关注前端新技术
- 参与开源项目
- 阅读技术博客
- 参加技术会议
